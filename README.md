# PsychoClient — Клиентское приложение сервиса Psycho

## Описание проекта

Графическое клиентское приложение для взаимодействия с сервисом записи к психологам. Реализовано на базе Qt с использованием многопоточной архитектуры для обеспечения отзывчивости интерфейса.

Проект выполнен в рамках дисциплины "Объектно-ориентированное программирование" и "Многопоточное программирование".

---

## Функциональность

### Авторизация и профиль
- Регистрация с динамической формой (скрытие полей для клиентов).
- Безопасный вход в систему.
- Редактирование личных данных и удаление аккаунта.

### Для клиентов
- Поиск психологов по имени и специализации.
- Просмотр подробной информации о специалисте.
- Бронирование свободных временных слотов.
- Личный кабинет со списком предстоящих сессий.
- Интерактивный календарь записей.
- Дневник самонаблюдения для личных заметок.

### Для психологов
- Управление собственным расписанием (добавление и удаление слотов).
- Просмотр списка пациентов и текущих сессий.
- Календарь занятости.

### Коммуникация
- Чат в реальном времени между клиентом и психологом.
- Система уведомлений о новых событиях.

## Технологии

- **C++17**
- **Qt 6.11.0** - GUI, Network, Core
- **QThread** - выделенный поток для сетевого взаимодействия
- **CMake** - система сборки
- **JSON** - формат обмена данными с сервером

---

## Многопоточность (NetworkThread)

В приложении реализовано строгое разделение потоков:
- **GUI Thread**: Основной поток, отвечающий за отрисовку интерфейса и взаимодействие с пользователем.
- **Network Thread**: Выделенный поток (`QThread`) для всех сетевых операций.
- Взаимодействие между потоками осуществляется через **сигналы и слоты** и `QMetaObject::invokeMethod`, что исключает блокировку интерфейса при сетевых задержках.

## Сборка и запуск

### Требования
- CMake 3.20+
- Qt 6.11.0 (MSVC 2022 64-bit)
- Компилятор с поддержкой C++17

### Сборка

```powershell
mkdir build
cd build
cmake ..
cmake --build . --config Release
```

### Запуск
```powershell
./Release/psycho_client.exe
```
